# Backend Section

This project uses **PostgreSQL** as the database, **pgAdmin 4** for database management, and **Express.js** as the backend server.

## Requirements

Before starting, make sure you have installed:

- PostgreSQL
- pgAdmin 4
- Node.js (LTS version)
- Git

---

# 1. Install PostgreSQL

1. Download PostgreSQL from the official website:
   https://www.postgresql.org/download/

2. Run the installer.

3. During the installation:
   - Select **PostgreSQL Server** and **pgAdmin 4**.
   - Choose an installation directory.
   - Create a password for the **postgres** user.
   - Leave the default port (**5432**).

4. Finish the installation.

---

# 2. Open pgAdmin 4

1. Launch **pgAdmin 4**.
2. Enter the password created during the installation.
3. Connect to the default PostgreSQL server.

---

# 3. Create a Database

1. In pgAdmin, expand **Servers → PostgreSQL**.
2. Right-click **Databases**.
3. Select **Create → Database**.
4. Enter the database name.
5. Click **Save**.
